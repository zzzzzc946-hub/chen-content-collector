## 更新规则

1. 合并影响用户可见行为、兼容性或发布资产的变更后，先判定是否升级版本。
2. 同时更新 `VERSION`、`CHANGELOG.md`、本表和 Release 资产名。
3. 只有已验证的 Git tag 才能标记为已发布；构建成功、合并完成或上传草稿均不等于已发布。
